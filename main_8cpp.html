Using the Boost library develop a utility to detect duplicate files.

Result list of full file paths with identical content, printed to standard output. One file per line. Identical files are output in one group. The different groups are separated by a empty line. Usage example: ./bayan -h ./bayan -i "./" -l 2 -m 1 –masks "bay(.*)" ./bayan -i "../" -S 8 -l 3 -m 1 –masks "test_test(.*)" ./bayan –iDir "./" –eDir "./1/2" './CMakeFiles' -l 6
